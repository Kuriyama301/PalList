# PalList

## 概要
PalList は、アルバイトの仕分け作業を効率化するための Web アプリです。
倉庫内での仕分け作業において、注文用紙から注文者の情報を素早く確認し、作業時間を短縮することを目的としています。

## 課題と解決策
### **課題**
- 仕分け作業時、複数の注文用紙（A4サイズ、表裏に4件ずつ記載）を確認する必要があり、該当の注文を探すのに時間がかかる。
- 現状ではページをめくりながら注文者の情報を確認しているため、作業効率が悪い。

### **解決策**
- **OCR（光学文字認識）技術を活用し、カメラで注文用紙を撮影すると注文者の名前を自動で抽出する機能を提供。**
- **すべての注文者名を一覧表示し、スクロールで簡単に確認できるようにする。**
- **OCR の誤認識を手動で修正できる機能を搭載。**

## 技術構成
- **バックエンド:** Ruby on Rails
- **フロントエンド:** React
- **データベース:** PostgreSQL
- **デプロイ:** Heroku
- **OCR エンジン:** 最適なものを選定予定

## 環境構築
### **前提条件**
以下の環境が必要です。
- Ruby 3.x
- Rails 7.x
- Node.js 16.x 以上
- PostgreSQL
- Yarn

### **セットアップ手順**
```sh
# リポジトリをクローン
git clone git@github.com:Kuriyama301/PalList.git
cd PalList

# 必要な gem のインストール
bundle install

# フロントエンドのセットアップ
yarn install

# データベースの作成
rails db:create
rails db:migrate

# サーバーの起動
rails s
